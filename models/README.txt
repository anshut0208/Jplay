GPT-2 Pretrained Weights Placeholder

This folder is for the GPT-2 model weights.

Steps to use:
1. Download GPT-2 small model from Hugging Face: https://huggingface.co/gpt2
2. Place all files inside this folder (config.json, pytorch_model.bin, merges.txt, vocab.json, tokenizer_config.json)
3. Then you can run ChatRec_Model.ipynb or generate_reply.py successfully.
